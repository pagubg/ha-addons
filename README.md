# HA Addons Repository

This repository contains Home Assistant addons developed and maintained by the community.

## Installation

1. Add this repository URL to your Home Assistant instance:
   - Go to Settings → Add-ons → Add-on Store
   - Click the three-dot menu and select "Repositories"
   - Add: `https://github.com/pagubg/ha-addons`

2. The addon will appear in the Add-on Store and can be installed from there

## Available Addons

### SCP Backup

Automatically backs up your Home Assistant configuration to a remote server via SCP (SSH). Supports manual triggering and scheduled transfers with SSH key authentication.

- **Features**: Scheduled transfers, manual backup creation, local backup retention policies, transfer verification
- **Installation**: Available in Add-on Store after adding this repository

For detailed information, see the [SCP Backup DOCS.md](scp-backup/DOCS.md)
